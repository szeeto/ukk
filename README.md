# Aplikasi Perpustakaan Sekolah Digital

Aplikasi ini adalah sistem manajemen perpustakaan berbasis web yang digunakan untuk mengelola data buku, anggota, dan transaksi peminjaman/pengembalian buku di lingkungan sekolah.

## Fitur Utama
- Login Admin dan Anggota
- Manajemen Data Buku (tambah, edit, hapus)
- Manajemen Data Anggota (tambah, edit, hapus)
- Peminjaman dan Pengembalian Buku
- Riwayat Transaksi Peminjaman
- Status ketersediaan buku otomatis

## Teknologi
- PHP (Native)
- MySQL
- Bootstrap (CSS Framework)

## Struktur Folder
- `admin/` : Halaman dan proses untuk admin
- `anggota/` : Halaman untuk anggota
- `css/` dan `js/` : Asset Bootstrap

## Cara Menjalankan
1. Clone/copy source code ke folder `htdocs` XAMPP
2. Import database `perpus` ke MySQL
3. Jalankan XAMPP (Apache & MySQL)
4. Akses melalui browser: `http://localhost/ukk/`

---
Aplikasi ini cocok untuk kebutuhan perpustakaan sekolah skala kecil-menengah.
